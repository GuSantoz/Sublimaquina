# Sublimaquina

Acesso: https://gusantoz.github.io/Sublimaquina/

Projeto desenvolvido para um amigo que possui uma loja de manutenção de impressoras. Utilizaremos em diversos trabalhos acadêmicos, desenvolvendo Frontend, Backend e também Banco de Dados em nuvem utilizando a AWS.
O projeto consome uma API desenvolvida por um professor e utiliza as linguagem HTML, CSS e JavaScript.
